बोलो। It types.

Hold a key, speak in any of 22 Indian languages (or the Hinglish you actually speak) and clean text appears wherever your cursor is. WhatsApp, email, code editor, anywhere on Windows.

Download for Windows Free and open source · Windows 10/11 · bring your own Sarvam AI key
Kal ki meeting 3 baje shift kar do, aur sabko email bhej dena

Made for the way Bharat speaks

Not English dictation with Hindi bolted on. Code-mixed speech is the first-class citizen here.

Hinglish that stays Hinglish

Speak naturally mixed Hindi-English; the output mirrors you. Prefer Devanagari or full English translation? One click.

22 Indian languages

Hindi, Tamil, Telugu, Bengali, Marathi, Kannada, Malayalam and more, auto-detected. Powered by Sarvam AI's Saaras model.

See your words live

A floating card types your words out while you're still speaking, then the text lands at your cursor in about a second.

Works in every app

Anywhere you can type, you can speak: browsers, WhatsApp Desktop, Word, VS Code, terminals.

Push-to-talk, no wake words

Hold Ctrl + Alt, talk, release. Esc cancels. Rebind to any keys in Settings.

Open source, your key

MIT licensed. Speech goes only to Sarvam's API with your own free key. No middleman servers, no subscription.

Running in two minutes

The installer handles everything; onboarding walks you through the one thing it can't do for you.

Install

Download and run BolFlow-Setup. No Python, no dependencies, no command line.

Paste your free key

Onboarding opens on first launch: grab a free API key from dashboard.sarvam.ai, paste it, and it's verified with a live call.

Speak

The built-in test box lets you try it immediately. Then it lives quietly in your tray.

About the SmartScreen prompt: BolFlow is a new, unsigned open-source app, so Windows may show "Windows protected your PC" on first run. Click "More info" → "Run anyway". The source is public on GitHub, so you can read every line or build it yourself.
